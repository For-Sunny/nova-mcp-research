## MCP Tools

- `Click-Tool`: Click on screen at coordinates
- `Type-Tool`: Type text on elements
- `Scroll-Tool`: Scroll windows/regions
- `Drag-Tool`: Drag from point to point
- `Move-Tool`: Move mouse pointer
- `Shortcut-Tool`: Press keyboard shortcuts
- `Wait-Tool`: Pause execution
- `State-Tool`: Capture desktop state with optional screenshot
- `App-Tool`: Launch, resize, switch applications
- **`Powershell-Tool`**: **UNRESTRICTED** - Execute ANY PowerShell command
- `Scrape-Tool`: Scrape webpage content
