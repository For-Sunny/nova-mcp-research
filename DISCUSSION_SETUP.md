**Description**: Technical questions about installing and using the MCP servers

**Initial Post Title**: "Installation Issues? Ask Here"
**Initial Post Content**:
```markdown
This category is for technical questions about installing, configuring, and using the MCP servers.

## Before Asking

1. Check the README for your edition (Basement Revolution or Enterprise Safe)
2. Verify you meet hardware requirements
3. Look for similar questions already answered

## When Asking

Include:
- **Edition**: Basement Revolution or Enterprise Safe?
- **Package**: Which MCP server (CASCADE, Faiss, Windows, File Server)?
- **OS**: Windows version, Linux distro, or macOS?
- **Error message**: Full error text (use code blocks)
- **What you tried**: Steps you've already taken

## What This Category Is NOT For

- Feature requests (those go in Issues)
- Bug reports (those go in Issues)
- "Can you build this for me?" (that's consulting)

We're a research community, not a support team. Help each other!
```
